# 智能文献管理系统 python1258

## 项目概述

智能文献管理系统是基于Python开发的，旨在提供文献资料的有效管理和注释功能。

## 技术栈

- **后端框架：** Django
- **编程语言：** Python 3.7/3.8
- **数据库：** MySQL 5.7 或更高版本（理论支持5.X系列）
- **数据库工具：** Navicat 11（推荐）
- **开发工具：** PyCharm、VS Code
- **前端框架：** Vue.js

## 功能模块

- 文献管理
- 文献注释
- 智能搜索
- 用户权限管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- Python 3.7/3.8
- Django
- MySQL 5.7+
- Vue.js

## 部署步骤

1. 安装Python运行环境
2. 安装MySQL数据库
3. 配置数据库连接
4. 安装依赖库
5. 运行后端服务
6. 构建并运行前端服务


## 核心亮点

- 基于Python，易于维护和扩展
- 使用Django框架，保障了系统的稳定性和安全性
- 支持智能搜索和文献注释，提高文献研究效率
- 灵活的用户权限管理，满足不同用户的需求
- 兼容多种数据库版本，灵活部署
